# Crime Data Analysis (Python)

## Project Overview
This project analyzes Washington, DC crime incident data from 2025 using Python to identify common offense types, district-level patterns, ward-level trends, and time-based crime activity.

## Dataset
This dataset comes from DC Open Data's Crime Incidents in 2025 resource.

## Analysis Performed
- Data cleaning and preprocessing
- Crime type frequency analysis
- Ward and district comparisons
- Shift-based crime analysis
- Monthly and weekly trend analysis
- Neighborhood and PSA hotspot analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- GitHub

## Files
- `Crime_incidents_in_2025.csv`
